# Windows Timer Example
A simple example of how to use this library to create and manage software timers in a Windows. Windows was selected because I do not feel like plugging in a microcontroller right now.

## Build

This was built on windows with gcc:

- `gcc main.c swTimer.c -o main.exe`

and executed using:

- `.\main.exe`

Not a complex example. Just wanted to show something working.
